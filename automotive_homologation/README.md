## Quick References

### Key Regulatory Authorities

| Authority | Country/Region | Role |
|---|---|---|
| DGTP / ARAI | India | Type Approval authority |
| KBA | Germany | German Federal Motor Transport Authority |
| VCA | UK | Vehicle Certification Agency |
| NHTSA | USA | FMVSS standards (Federal Motor Vehicle Safety Standards) |
| UNECE WP.29 | Geneva | UN Working Party — ECE Regulations source |
| TRL / Applus IDIADA | EU | Technical Services for TA testing |

### Most Critical ECE Regulations for Lighting

| Regulation | Scope | Key Test |
|---|---|---|
| R112 | Motor vehicle headlamps with asymmetric beam | Photometric grid, beam pattern, glare |
| R7 | Position, stop, end-outline lamps | Luminous intensity in candela (cd), colour |
| R6 | Direction indicators (turn signals) | Flash rate, intensity, colour |
| R48 | Installation of lighting on vehicles | System-level compliance, spacing, height |
| R10 | EMC (Electromagnetic Compatibility) | Radiated/conducted emissions from lamps |
| R119 | Cornering lamps | Activation conditions, beam pattern |
| R123 | AFS (Adaptive Front Lighting Systems) | Mode switching, bending light |
