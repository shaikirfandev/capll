## Interview Strategy

### Before the Interview
- Read the **Job Description** carefully; map each JD requirement to your experience (use Project Walkthroughs file)
- Practice **Self Introduction** aloud (2-min version is most common)
- Review **Technical Q&A** for the specific domain the company works in
- Prepare **2 STAR stories** per project (BYD, BMW, Lexus)

### During the Interview
- Start with the **structured self-introduction** (don't freestyle)
- For every answer, use: **Context → Action → Result (with numbers)**
- When asked about career break: use the **prepared script** (confident, brief, forward-looking)
- Ask **2–3 smart questions** at the end

### Your Unique Selling Points (highlight these)
1. **Multi-domain coverage**: ADAS + Infotainment + Telematics + Cluster (rare combination)
2. **Automation impact**: 30–35% efficiency gains through CAPL scripting
3. **Diagnostic depth**: Hands-on UDS (0x10, 0x19, 0x22, 0x27, 0x2E) across all ECU types
4. **OEM exposure**: BYD (EV leader), BMW (premium), Lexus (luxury) — 3 top-tier OEMs
5. **V-Model understanding**: Requirements (DOORS) → Test Design → Execution → Defect closure
